### 大语言模型是大脑——但是仅靠大脑无法做事
现在的大多数人都熟悉LLM，像是ChatGPT或者Gemini.他们在推理，总结和交流方面表现的很好。他们通过语言来思考。

但是有一个致命的缺陷：仅靠他们自己，LLMs不能真正的做任何事情

在没有额外工具的情况下，它们不能运行脚本，点击按钮，修改文件或者改变系统状态，一个大语言模型可以将一个行动描述的很好，但却不能触碰到真实世界。

理解和行动之间的鸿沟准确的解释了为什么智能代理会存在。

### 智能代理：当语言模型拥有手的时候
智能代理可以被看做装备有各种工具和控制循环的LLM。

这些工具可能包含APIs,文件系统，浏览器，脚本或者是操作系统的行为。当这些工具被包含，该系统就能将解释转变为行动。

我们早就在不同的环境中看到过这些模式。
- 像是IBM Bob系统通过一个基于GUI的集成开发环境来操作
- Claude Code 居住在开发者终端中

它们都能被视为智能代理样式的系统——主要是为了科技用户而设计。
